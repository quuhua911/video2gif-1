video2gif
=========

Convert a video file (mp4, mov, etc.) to a reasonable GIF.

Usage
-----

Requires ffmpeg.

```bash
$ video2gif foo.mp4
$ video2gif foo.mp4 foo.gif --scale 640
$ video2gif foo.mp4 foo.gif --fps 24 --scale 50%
```
